HashMap<Keys,Value>


CREATE <collection-name> ON-KEY <data-field>
DELETE <collection-name>
SELECT <collection-name>
	INSERT <data-structure>
	SEARCH <data-field> <operand> <data-value> AND <data-field> <operand> <data-value>
	REMOVE <data-value>


Constrains:
A) <collection-name>, <data-field>, and all String <data-value> must be always quoted (like "property").
B) <collection-name> and <data-field> cannot have whitespaces characters.
C) <data-structure> must be a string json formatted (are excluded by the constraints A).

Examples:
1)
CREATE domains ON-KEY name
2)
CREATE operations ON-KEY id
3)
CREATE users ON-KEY id
4)
DELETE operations ON-KEY id
5)
